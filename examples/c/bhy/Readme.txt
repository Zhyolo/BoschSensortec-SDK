To build any of the examples ,go to the folder and run 

mingw32-make SHUTTLE_BOARD=<BHxxxx>

where <BHxxxx> can be 
	BHI160
	BHI160B
	BHA250
	
Refer the shuttle board for this information


Running any example application will upload a RAM patch.
Wait for a few seconds for the RAM patch to upload.
Try to move the board randomly and watch the example application in action.
